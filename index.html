Zeal

Zeal is a simple offline API documentation browser inspired by Dash (OS X app), available for Linux and Windows.

  • Quickly search documentation using Alt+Space (or customised) hotkey to display Zeal from any place in your workspace.
  • Search in multiple sets of documentation at once.
  • Don't be dependend on your internet connection.

Documentation included

Currently the following docsets are directly downloadable in Zeal:

  1. Django, jQuery, Ruby 1 and 2 - thanks to Dash's docset exchange programe. Now also author of Dash has agreed to provide all of his docsets to Zeal users, which at the moment can be downloaded manually from Dash's website. Direct downloads from Zeal are to be implemented in future, though no specific plans for that exist yet (see issue #24).
  2. Appcelerator Titanium, ExtJS 4.1, Python 2.7.3, Sencha Touch 2.1, Qt 5 - generated for Zeal.

You can also generate your own docsets using the same format Dash uses - see instructions on Dash's website.

Open source!

Zeal is released under the GPL license, which allows anyone to contribute changes. And it is available free of charge.
