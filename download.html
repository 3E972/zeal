Download Zeal

There are various ways to download Zeal, depending on which operating system you use, and whether you want to use precompiled binary, or build it yourself from source code.

Windows binary

A 32-bit zipped Windows binary with all dependencies included is available to download from Bitbucket.
Simply decompress the .zip file and run zeal.exe from any place.

Download zeal-20130803-2.zip (33MB) »

Ubuntu Linux PPA

If you use Ubuntu Linux Precise (12.04), Quantal (12.10), or Raring (13.04), you can easily install Zeal from its PPA.

To install it:

  1. Run add-apt-repository commands listed on the PPA page
  2. then do apt-get update,
  3. and apt-get install zeal
After installing Zeal can be started from command line by running zeal.
Go to Ubuntu PPA »

Arch Linux AUR

A package in AUR (zeal-git) is available for Arch Linux users, thanks to Whyme Lyu.

See zeal-git in AUR »

Source code

You can also compile Zeal from source code, assuming you have Qt >= 5.0.0 development tools installed. Clone the github repository linked below, and run qmake and make from the zeal subdirectory.

See GitHub project page »

Usage

After installing/compiling, to quickly download documentation sets, click 'Edit', 'Options', 'Docsets', 'Download...'.
